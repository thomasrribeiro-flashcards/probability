# probability

Flashcards for Probability Theory — for spaced repetition review via [flashcards](https://github.com/thomasrribeiro/flashcards).

## Coverage

| File | Topic |
|------|-------|
| 01_combinatorics | Counting, permutations, combinations, binomial/multinomial theorem, stars and bars |
| 02_axioms_events | Sample space, events, Kolmogorov axioms, inclusion-exclusion |
| 03_conditional_bayes | Conditional probability, independence, law of total probability, Bayes |
| 04_discrete_random_variables | PMF, CDF, expectation, variance, LOTUS |
| 05_discrete_distributions | Bernoulli, binomial, geometric, negative binomial, hypergeometric, Poisson |
| 06_continuous_random_variables | PDF, CDF, expectation for continuous RVs, quantiles |
| 07_continuous_distributions | Uniform, exponential, normal, gamma, beta |
| 08_joint_distributions | Joint/marginal/conditional distributions, independence, iid |
| 09_expectation_covariance | Linearity, covariance, correlation, Cauchy-Schwarz |
| 10_transformations | $Y=g(X)$, change of variables, Jacobian, convolution, order statistics |
| 11_mgfs | Moment generating functions, uniqueness, sums of independent RVs |
| 12_limit_theorems | Markov/Chebyshev, LLN, CLT, modes of convergence |
| 13_conditional_expectation | Conditional expectation as RV, tower property, Markov chains intro |
